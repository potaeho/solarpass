# SolarMap PRD — 제품 요구사항 문서
## 제품 목적
태양광 발전 사업자, 투자자, 개인 설치 희망자가 **원하는 지역의 태양광 설치 가능 여부를
즉시 파악하고, 인허가 전 과정을 플랫폼 안에서 완결**할 수 있도록 한다.
---
## 사용자 시나리오
```
1. 앱 실행 → 전국 지도 확인
2. 지도에서 관심 지역 탭 (또는 검색)
3. 해당 지역의 5가지 기준별 가능 여부 확인
4. "인허가 진행하기" 탭
5. 7단계 인허가 프로세스 확인
6. 각 단계별 전문 업체 선택 (입찰 또는 즉시 선택)
7. 3단계(개발행위허가) 시 토목사무소 자동 매칭 → 연락처 연결
8. 진행 현황 탭에서 단계별 상태 + 예상 기간 + 병행 가능 여부 모니터링
```

## 기능 1. 지도 홈 화면
### 1-1. 지도 레이어
**지도 라이브러리**: `react-native-maps`
**레퍼런스 UX**: 네이버 지도 앱 (상단 검색창, 우측 컨트롤, 하단 시트)
지도 위에 지역별 가능 여부를 원형 마커로 표시한다.
| 색상 | 의미 | FeasibilityStatus |
|------|------|-------------------|
| 초록 (#19A050) | 설치 가능 | `available` |
| 노란 (#FAB400) | 조건부 가능 | `conditional` |
| 빨간 (#DC3232) | 설치 불가 | `unavailable` |
마커 동작:
- 탭 시 하단 시트(BottomSheet) 올라옴 — 미리보기 카드 표시
- 핀치 줌에 따라 마커 크기 자동 조절
- 같은 권역의 마커가 겹치면 클러스터링 처리
**상단 검색창** (네이버 지도 스타일):
- placeholder: "지역, 주소 검색"
- 탭 시 검색 화면으로 push
- 검색 결과: 지역명, 행정구역 단위 (시/군/구)
**우측 플로팅 버튼** (세로 스택):
- 줌인 / 줌아웃
- 현재 위치
- 레이어 전환 (위성/일반)
**좌하단 범례**:
```
● 설치 가능  ● 조건부 가능  ● 설치 불가
```
### 1-2. 하단 시트 (지역 선택 시)
`@gorhom/bottom-sheet` 사용. 스냅 포인트: `['30%', '70%']`
**30% 상태 (미리보기)**:
```
[지역명]                    [뱃지: 설치가능/조건부/불가]
한전 여유 선로 OOMWh · 이격거리 충족
                    [상세 보기 →]
```
**70% 상태 (상세)**:
5가지 체크리스트 + 진행하기 버튼

## 기능 2. 지역 상세 — 5가지 판단 기준
화면: `app/region/[id].tsx`
각 기준은 **pass / warning / fail** 상태를 가진다.
### 실발전 데이터 패널 (한국전력거래소 공공데이터 기반)
지역 상세 화면 최상단에 실발전 실적 카드를 표시한다.
데이터 출처: 한국전력거래소 지역별 시간별 태양광 발전량 (2025년 연간)
```typescript
type RegionSolarStats = {
  region: string;             // "전라남도"
  annualGenerationMWh: number; // 연간 발전량 (MWh)
  nationalRank: number;        // 전국 순위 (1~17)
  totalRegions: number;        // 17
  installedCapacityMW: number; // 설치 설비용량 (MW) — 2024년 기준
};
```

발전 실적 등급 기준:
```
최우수: 연간 100만 MWh 이상      → 마커 가장 진한 초록
우수:   연간 50~100만 MWh        → 마커 진한 초록
양호:   연간 20~50만 MWh         → 마커 중간 초록
보통:   연간 5~20만 MWh          → 마커 연한 초록
낮음:   연간 5만 MWh 미만         → 마커 가장 연한 초록
```
> 지도 마커 색상: 설치 가능 여부(초록/노랑/빨강) + 발전 실적 등급(진하기)으로 2차원 표현
> 예) 설치 가능(초록) + 최우수 발전 실적 → 가장 진한 초록 마커
**전국 태양광 설비용량** (2024년 12월 31일 기준): **230,752 MW**
### 기준 1: 토지이용계획열람
```
판단 항목:
- 용도지역 (계획관리지역 / 농림지역 / 보전관리지역 등)
- 용도구역 (군사보호구역, 절대보전지역, 수변구역 등)
- 토지이용규제 (개발제한구역, 산지관리법 적용 여부)
pass   → 설치 가능 용도지역, 규제 없음
warning → 농림지역 일부 포함 등 조건부
fail   → 군사보호구역, 절대보전지역, 개발제한구역
```
### 기준 2: 한전 계통망 용량
**반드시 수치로 표시해야 하는 항목**:
```
표시 항목:
- 여유 선로 용량: OO MW
- 계통 용량 사용률: OO% (프로그레스 바)
- 접속 대기 건수: OO건
- 예상 접속 대기 기간: OO개월
pass   → 여유 선로 20MW 이상, 대기 없음
warning → 여유 선로 5~20MW, 또는 대기 있음
fail   → 여유 선로 5MW 미만, 포화 상태
```
UI 컴포넌트 `GridCapacityBar`:
```
계통 용량 사용률 88% · 여유 8MW
[████████████████░░] 88%
접속 대기 3건 · 예상 대기 4개월
```
### 기준 3: 이격거리 확인
```
판단 항목:
- 주거지역 이격거리 (지자체마다 다름, 보통 100~300m)
- 도로 이격거리 (50~100m)
- 학교·병원 이격거리 (200~500m)
- 문화재 이격거리 (500m)
pass   → 모든 이격거리 충족
warning → 일부 이격거리 기준 근접 (여유 10% 이내)
fail   → 이격거리 미충족
```
### 기준 4: 전국 공통 법규
```
검토 법령:
- 산지관리법 (보전산지 여부)
- 농지법 (농업진흥구역 여부)
- 환경영향평가법 (환경민감지역)
- 전기사업법 (발전사업 허가 기준)
- 신에너지법 (설비 인증 기준)
pass   → 모든 법령 이상 없음
warning → 조건부 허용 지역 (환경평가 필요 등)
fail   → 법령 위반 (보전산지, 농업진흥구역 등)
```
### 기준 5: 지자체 조례
```
검토 항목:
- 지자체별 추가 이격거리 기준 (전국 기준 외 가산)
- 태양광 허가 불가 구역 지정 여부
- 경관심의 대상 여부
- 주민 동의 요건
pass   → 조례 충족, 추가 기준 없음
warning → 추가 이격기준 적용, 주민동의 필요
fail   → 조례상 허가 불가 구역
```

## 기능 3. 인허가 프로세스
화면: `app/permit-flow/[regionId]/index.tsx`
### 인허가 7단계 정의
> **단계 간 연계 관계 — 반드시 UI에 표현해야 함**
>
> | 관계 유형 | 해당 단계 | 설명 |
> |-----------|-----------|------|
> | 독립 병행 | 2단계 ↔ 3단계 | 발전사업허가·개발행위허가는 완전 독립 — 둘 다 통과해야 진행 가능 |
> | 병행 가능 | 3·4·5단계 | 동시 병행 진행 가능 (단, 5단계 환경영향평가 통과가 3·4단계의 전제) |
> | 선행 의존 | 1단계 → 7단계 | 계통연계는 1단계에서 비공식 타진, 2단계에서 접속 의향서 수령 필수 |
> | 착공 조건 | 2~5단계 → 6단계 | 2·3·4·5단계 전부 완료 후에야 6단계 착공 가능 |
```
Step 1. 사전 부지검토
  - 성격: 사업자 자체 수행 (인허가 주체 없음)
  - 예상 기간: 1~4주
  - 핵심 확인 항목:
      · 토지이음(eum.go.kr) — 지목·용도지역·규제 확인
      · 환경공간정보서비스(egis.me.go.kr) — 생태·자연도 등급
      · 대법원 인터넷등기소 — 소유권·가압류·근저당
      · 자치법규정보시스템 — 이격거리·경관 조례
      · 한전 배전운영처 비공식 접촉 — 선로 여유 용량 (최우선)
      · 에너지공단 자원지도 + 드론 측량 — 일사량·경사도
  - 추천 업체: 그랜드썬기술단, 젠코, G2P, 정의행정사
  - 핵심 리스크: 계통 불가 지역 진입 시 투자 전액 손실
Step 2. 발전사업허가 [독립 병행]
  - 인허가 주체: 용량별 상이
      · 500kW 이하 → 시장·군수·구청장
      · 500kW 초과~3MW → 시·도지사
      · 3MW 초과 → 기후에너지환경부
  - 근거 법령: 전기사업법 제7조
  - 예상 기간: 50일 (보완 시 90일+)
  - 필요 서류: 발전사업허가 신청서, 사업계획서, 단선결선도,
               발전원가명세서, 한전 접속 의향서, 재원조달계획서
  - 추천 업체: OCI파워, 신성이엔지, SK E&S 엔솔브, G2P, 젠코
               (3MW 이상) 법률사무소 솔라리스, 법무법인 지평
  - 핵심 리스크: 허가 취득 후 3년 내 미착공 시 자동 취소
Step 3. 개발행위허가 [독립 병행 / 병행 가능]
  - 인허가 주체: 시장·군수·구청장 (도시계획과·건축과·허가과)
  - 근거 법령: 국토의 계획 및 이용에 관한 법률 제56조
  - 예상 기간: 법정 15일 / 실무 수개월~수년
  - 토목사무소 연결 대상 단계 (시청/군청 앞 토목사무소 자동 매칭)
  - 필요 서류: 신청서, 형질변경 계획서, 토지 소유권 증명,
               배치도, 구조계산서, 경관 심의 결과서
  - 추천 업체: 정의행정사, 도시·건축 설계사무소, KTS건설, G2P, 젠코
  - 핵심 리스크: 주민 반대 시 무기한 보류 / 지자체 재량 광범위
Step 4. 산지·농지전용 [병행 가능]
  - 인허가 주체: 용지 유형별 상이
      · 산지전용 소규모 → 시장·군수 (산림과)
      · 산지전용 대규모 → 지방산림청장
      · 농지전용 진흥구역 외 → 시장·군수 (농지과)
      · 농지전용 진흥구역 내 → 농림축산식품부장관 (사실상 불가)
      · 영농형 일시사용 → 시장·군수 (농지과)
  - 근거 법령: 산지관리법 제14조, 농지법 제34조
  - 예상 기간: 15~60일
  - 추천 업체: 산림기술사 사무소, 정의행정사, 그랜드썬기술단, G2P
  - 핵심 리스크: 보전산지·농업진흥구역 전용 불가
                 복구비용 현금 예치 부담
                 영농형 8년 만료 후 연장 불허 시 원상복구 의무
Step 5. 환경영향평가 [병행 가능 / 3·4단계 전제 조건]
  - 인허가 주체:
      · 5,000㎡~33만㎡ 미만 → 지방환경청장 (소규모 환경영향평가)
      · 33만㎡ 이상 → 환경부장관 (전략환경영향평가)
  - 근거 법령: 환경영향평가법
  - 예상 기간: 3~12개월 (보완 반복 시 1년+)
  - 추천 업체: EIASS 등록 환경영향평가 전문 대행업체만 가능
  - 핵심 리스크: 생태·자연도 1~2등급 포함 시 협의 불가 → 사업 포기
                 평가서 작성 비용 수천만~1억 원 이상
Step 6. 공사계획 신고 및 시공
  - 인허가 주체:
      · 공사계획 신고 → 산업통상자원부 위임기관 KESCO (전기안전여기로)
      · 착공 신고 → 시장·군수·구청장 (건축과·허가과)
  - 근거 법령: 전기사업법 제61조, 전기안전관리법
  - 예상 기간: 1~6개월
  - 착공 조건: 2·3·4·5단계 전부 완료 후 착공 가능
  - 추천 업체: 그랜드썬기술단, 신성이엔지, OCI파워 (EPC)
               한전기술(KEPCO E&C) (감리)
  - 핵심 리스크: 무면허 전기공사 시 준공검사 불가
                 공사 중 주민 민원 → 공사 중단 명령
Step 7. 계통연계 및 사용전검사
  - 인허가 주체 (4개 기관 동시 대응):
      · 계통 접속 신청 → 한국전력공사(KEPCO) 배전운영처
      · 사용전검사 → 한국전기안전공사(KESCO) (전기안전여기로)
      · 전력거래 등록 → 한국전력거래소(KPX) — 거래 6개월 전 신청 필수
      · REC 설비 확인 → 한국에너지공단 — 사용전검사 후 1개월 이내 필수
  - 근거 법령: 전기사업법 제67조, 전기안전관리법
  - 예상 기간: 3~24개월 (선로 공사 대기 포함)
  - 추천 업체: 그랜드썬기술단, 신성이엔지, OCI파워 (EPC)
               법률사무소 솔라리스 (REC·PPA·FIT 계약)
  - 핵심 리스크: 선로 용량 부족 = 접속 불가 → 사업 좌초 (전체 최대 리스크)
                 LVR 추가 비용 수천만~수억 원
                 REC 설비 확인 1개월 기한 미준수 시 수익 소급 불가
```
### 단계별 병행 가능 여부 — UI 표현 방식
```
타임라인에서 병행 가능 단계는 같은 행(row)에 나란히 표시:
[1단계] 사전 부지검토 ─────────────────────────────→
        ↓
[2단계] 발전사업허가 ──────────────────────→  ┐ 독립 병행
[3단계] 개발행위허가 ──────────→  ┐           │ (둘 다 완료 필수)
[4단계] 산지·농지전용 ─────────→  ├ 병행 가능  │
[5단계] 환경영향평가 ──────────────────────→  ┘           ┘
        ↓ (2~5단계 전부 완료 후)
[6단계] 공사계획 신고 및 시공 ────────────────────→
        ↓
[7단계] 계통연계 및 사용전검사 ─────────────────────→
```
### 업체 추천 시스템
업체 리스트는 **자체 평가 점수** 기반으로 정렬한다.
```typescript
// 추천 점수 계산 (100점 만점)
score =
  successRate * 0.5        // 허가 성공률 50%
  + regionMatch * 0.2      // 해당 지역 경험 20%
  + responseSpeed * 0.15   // 평균 응답 속도 15%
  + userRating * 0.15      // 사용자 평점 15%
```
업체 카드 UI:
```
[★ 94점]  그린허가법인 나주지사        성공률 94%
          전남 지역 전문 · 평균 응답 2시간
          [견적 요청]  [바로 연락]
```
업체 참여 방식: 입찰(견적 요청) 또는 즉시 신청 선택 가능

## 기능 4. 진행 현황 대시보드
화면: `app/(tabs)/status.tsx`
### 전체 진행률
```
전체 진행률                    1 / 7 단계
[███░░░░░░░░░░░░░░░░░] 14%
예상 잔여 기간: 약 12~24개월
```
### 단계별 타임라인
각 단계는 아래 상태 중 하나:
| 상태 | 표시 | 색상 |
|------|------|------|
| `completed` | ✓ 완료 | 초록 |
| `in_progress` | ◷ 진행중 | 파랑 |
| `parallel` | ⇌ 병행중 | 보라 |
| `pending` | ○ 대기 | 회색 |
| `delayed` | ⚠ 지연 | 빨강 |
타임라인 상세 (병행 단계 표현):
```
✓  1단계  사전 부지검토          완료 (2024.08.20)
◷  2단계  발전사업허가           진행중 · D-30일 예상  ┐ 독립 병행
⇌  3단계  개발행위허가           병행중               ┘
   4단계  산지·농지전용          ─ 대기 (3단계 착수 후 병행 시작)
   5단계  환경영향평가           ─ 대기
○  6단계  공사계획 신고·시공      대기 (2~5단계 완료 후)
○  7단계  계통연계·사용전검사     대기
```
### 병행 단계 안내 배너
```
ℹ 2단계(발전사업허가)와 3단계(개발행위허가)는
  동시에 진행할 수 있습니다. 두 단계 모두 완료되어야
  다음 단계로 진행됩니다.
```
### 알림 설정
- 단계 완료 시 푸시 알림
- 서류 보완 요청 시 알림
- 예상 기간 초과 시 알림
- REC 설비 확인 마감 D-7, D-1 알림 (7단계 사용전검사 완료 후 자동 트리거)
