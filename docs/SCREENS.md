# SolarMap SCREENS — 화면별 스펙
## 내비게이션 구조
```
Root Stack
├── (tabs)                          ← 탭 바 (하단 5탭)
│   ├── index          /            지도 홈
│   ├── permit         /permit      인허가 현황
│   ├── status         /status      진행현황
│   ├── office         /office      토목사무소
│   └── my             /my          MY
│
├── region/[id]        /region/:id  지역 상세
│
└── permit-flow/[regionId]
    ├── index          인허가 단계 목록
    ├── step/[stepId]  단계별 업체 선택
    └── office         토목사무소 연결
```
---
## 탭 바 구성
```
[지도 🗺]  [인허가 📋]  [진행현황 📊]  [토목사무소 🏢]  [MY 👤]
```
활성 탭: `#03C75A` (브랜드 초록) / 비활성: `#999999`

카드 배경색: `#FFFBEA` (연한 노란 — 태양 느낌), 테두리: `#FAB400`

### 체크리스트 아이콘 규칙
| status | 아이콘 | 배경색 | 아이콘색 |
|--------|--------|--------|---------|
| pass | checkmark-circle | #E6F7EE | #0D7A38 |
| warning | alert-circle | #FFF8E1 | #8A6500 |
| fail | close-circle | #FDECEA | #B71C1C |
### FeasibilityBadge
| status | 텍스트 | 배경 | 텍스트색 |
|--------|--------|------|---------|
| available | 설치 가능 | #E6F7EE | #0D7A38 |
| conditional | 조건부 가능 | #FFF8E1 | #8A6500 |
| unavailable | 설치 불가 | #FDECEA | #B71C1C |
### 불가 지역 처리
`feasibilityStatus === 'unavailable'`일 때:
- "인허가 진행하기" 버튼 비활성화 (회색, 비활성)
- 하단에 사유 요약 텍스트 표시: "군사보호구역 해당으로 인허가 진행이 불가합니다"

## 애니메이션 가이드
- BottomSheet 열기/닫기: `@gorhom/bottom-sheet` 내장 스프링
- 화면 전환: Expo Router 기본 슬라이드
- 탭 전환: 즉시 (애니메이션 없음)
- 프로그레스 바: `Animated.timing` 1.2초, easing: `Easing.out(Easing.cubic)`
- 마커 탭 피드백: `scale(0.9)` → `scale(1)` 100ms
